# Icon button

Use the following CSS properties for theming Puck's icon buttons.

```css showLineNumbers
.Container {
  --puck-iconbutton-primary-radius: 9999px;
  --puck-iconbutton-color-icon: #1f2937;
  --puck-iconbutton-color-bg-hover: #eef2ff;
  --puck-iconbutton-space: 16px;
}
```

## CSS properties

Defaults shown as `var(--puck-...)` reference [global tokens](/docs/api-reference/theming/global). See [computed values](/docs/api-reference/theming/computed-values#icon-button) for the values they resolve to.

| Token                                   | Default                                       | Description             |
| --------------------------------------- | --------------------------------------------- | ----------------------- |
| `--puck-iconbutton-space`               | `var(--puck-space-1)`                         | Padding                 |
| `--puck-iconbutton-color-bg-hover`      | `var(--puck-color-interactive-neutral-hover)` | Background on hover     |
| `--puck-iconbutton-color-bg-active`     | `var(--puck-color-interactive-soft)`          | Background when pressed |
| `--puck-iconbutton-color-icon`          | `currentColor`                                | Icon color              |
| `--puck-iconbutton-color-icon-hover`    | `var(--puck-color-interactive)`               | Icon color on hover     |
| `--puck-iconbutton-color-icon-active`   | `var(--puck-color-interactive)`               | Icon color when active  |
| `--puck-iconbutton-color-icon-disabled` | `var(--puck-color-text-subtle)`               | Disabled icon color     |

### Variants

**Primary variant (`--puck-iconbutton-primary-*`)**

| Token                                | Default                | Description   |
| ------------------------------------ | ---------------------- | ------------- |
| `--puck-iconbutton-primary-color-bg` | `transparent`          | Background    |
| `--puck-iconbutton-primary-radius`   | `var(--puck-radius-m)` | Border radius |

**Secondary variant (`--puck-iconbutton-secondary-*`)**

| Token                                      | Default                            | Description   |
| ------------------------------------------ | ---------------------------------- | ------------- |
| `--puck-iconbutton-secondary-color-bg`     | `var(--puck-color-surface)`        | Background    |
| `--puck-iconbutton-secondary-color-border` | `var(--puck-color-border)`         | Border color  |
| `--puck-iconbutton-secondary-border-width` | `var(--puck-border-width-regular)` | Border width  |
| `--puck-iconbutton-secondary-radius`       | `var(--puck-radius-round)`         | Border radius |
