# Theming

CSS properties for theming the default Puck user interface.

```css showLineNumbers
@import "@puckeditor/core/puck.css";

:root {
  --puck-color-interactive: #0f5fff;
  --puck-color-focus-ring: #3b82f6;
}
```

## Component theming

Component-level tokens override global tokens for specific components.

- [Action bar](theming/action-bar): The action bar shown when selecting a component.
- [Button](theming/button): The buttons, such as the Publish button.
- [Canvas](theming/canvas): The canvas preview area.
- [Drawer](theming/drawer): The component drawer.
- [Fields](theming/fields): The puck [fields](/docs/api-reference/fields).
- [Header](theming/header): The editor header.
- [Icon button](theming/icon-button): The icon buttons.
- [Outline](theming/outline): The outline panel.
- [Plugin bar](theming/plugin-bar): The side navigation for switching plugins.
- [Sidebar](theming/sidebar): The left and right sidebar panels.
- [Slot](theming/slot): The slots.
